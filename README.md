# 🎯 QuizMaster - React Quiz App

An interactive and responsive Quiz Application built using React and Vite.

## ✨ Features

* Welcome screen with player name input
* 5 Multiple Choice Questions
* 15-Second Timer for each question
* Progress Bar
* Correct and Incorrect Answer Indicators
* Final Score Calculation
* Personalized Result Screen
* Play Again Functionality
* Dark Mode Toggle
* Responsive User Interface

## 🛠️ Technologies Used

* React
* JavaScript (ES6+)
* CSS3
* Vite

## 📚 Concepts Practiced

* React Components
* useState Hook
* useEffect Hook
* Props
* Event Handling
* Conditional Rendering
* Controlled Forms
* Dynamic Styling
* State Management
